# SuperCurve

It's a 3D-style pong game written in pure Javascript.

![](multi/images/supercurve_ss_280.jpg)

# Deploy

## Frontend

Push the commits and GH Actions will deploy to Github Pages

## Highscores

```bash
cd workers
npm run deploy
```

## Multi server

```bash
fly deploy
```
